# 01_PLAN.md

본 문서는 'Three.js Tutorial Dashboard' 프로젝트의 기획 및 방향성을 기술합니다.

## 1. 프로젝트 비전
사용자가 3D 웹 그래픽의 원리를 실시간 렌더링 화면과 실제 코드를 대조해 보며 직관적으로 학습할 수 있는 '프리미엄 인터랙티브 학습 환경'을 제공합니다.

## 2. 목표
- **학습 편의성**: Three.js의 복잡한 속성 변화를 코드 수정 없이 Leva 컨트롤러를 통해 즉각 확인.
- **코드 접근성**: 렌더링 중인 예제의 실제 소스 코드를 별도 편집기 없이 웹 대시보드 내에서 즉시 확인.
- **현대적 아키텍처**: Next.js 15와 Tailwind CSS v4를 활용한 확장성 있는 프로젝트 구조 확보.

## 3. 타겟 사용자
- Three.js 및 React Three Fiber를 시작하는 초보 개발자.
- 3D 웹 디자인 시스템을 효율적으로 관리하고자 하는 프론트엔드 엔지니어.

## 4. 주요 로드맵
1. **MVP (Minimum Viable Product)**: Next.js 환경에서 기본 Geometry 렌더링 및 코드 뷰어 통합.
2. **Expansion**: 기존 Material, Light, Loader 예제 전체 마이그레이션.
3. **Premium UX**: 다크 모드, 부드러운 전환 효과, 모바일 대응 등 UI/UX 고도화.

## 5. 기대 효과
- 단순 예제 실행을 넘어, 코드를 직접 확인하고 속성을 조절하며 Three.js의 핵심 개념을 더 깊이 이해할 수 있음.
- 최신 Next.js 기술 스택을 적용한 고성능 3D 웹 애플리케이션의 모범 사례 구축.
